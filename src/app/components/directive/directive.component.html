Directive

Directives add additional behavior to elements in your templates.

Three different types of directives:

  • Components
    • Directives with a template.
  • Attribute directives
    • Directives that change the appearance or behavior of an element, component, or another directive.
  • Structural directives
    • Directives that change the DOM layout by adding and removing DOM elements.

NgModel

NgModel allows you to display a data property and update that property when the user makes input changes.

input1: {{ input1 }}

inputAndEmoji: {{ inputAndEmoji }}

NgIf

NgIf adds or removes an element.

NgFor

NgFor presents a list.

  • {{ name.firstName }} "{{ name.quoteName }}"

NgSwitch

NgSwitch displays one element from among several possible elements, based on a switch condition.
